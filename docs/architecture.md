# Kien truc Phase 1

## Frontend

ReactJS + Vite dung JSX. Giao dien dau tien gom dashboard va login placeholder.

## Backend

Stack da thong nhat:

```text
Backend: JavaScript ES Module + Node.js + Express.js
Database: MySQL + mysql2/promise
Authentication: JWT + bcrypt
Validation: Zod
Authorization: Express RBAC Middleware
Testing: Jest + Supertest
AI Integration: Axios goi Python/FastAPI
```

Kien truc backend:

```text
Route -> Middleware -> Controller -> Service -> mysql2/promise -> MySQL
```

Phase 1 moi co module `health`. Cac module nghiep vu se them tu Phase 2.
